# G80 M3 Car Wash Reference
**Isle of Man Green | Ceramic Coated | S58**

> Quick-lookup for dilutions, chemical notes, and wash reminders.  
> Full interactive checklist: open `index.html` in a browser.

---

## ⚠️ Global Rules

- **No wax or sealant** over the ceramic coating
- **Touchless drying on paint** — leaf blower only, zero towel contact
- **Cool panels always** — no washing in direct sun or on a hot car
- **Never use paint mitts on wheels** — three buckets, strict separation
- **Reset 1:9 is a foam cannon ratio** — does NOT apply to IK E Foam sprayers

## 🔄 Wash Mode Overview

### Bi-Weekly Normal Wash (18 steps)
Pre-treat bugs & tar → Pre-rinse wheels → MWC on wheels → ReTyre tires → APC wheel wells → Iron X on paint → Reset pre-foam → 2-bucket wash → Sheeting rinse → Blower dry → Exhaust tips → Exterior glass → Interior glass → Darkside tires → Trim dressing → Final inspection

### Ceramic Maintenance Wash (2×/year, adds 3 steps)
Inserts **after Iron X**:
1. **BH Touchless** via IK E Foam 12 Pro (60–90ml / 6L, dwell 1–5 min, rinse)
2. **Descale by hand** 1:100, one panel at a time

Inserts **after blower dry**:
3. **HydrO2 Foam** via IK E Foam 12 Pro (per label, flash 30–60 sec, blow off)

## ⚙️ Engine Bay Clean *(Optional — standalone session)*

**Fully cool engine required (2–3 hrs minimum)**

### Cover before ANY water contact
| Must bag + tape | Avoid direct spray (wipe only) |
|---|---|
| Alternator | ECU / DME modules |
| Fuse box / PDU (driver's side rear) | Coil pack connectors |
| Air intake snorkel & filter housing | Brake master cylinder reservoir cap |
| Throttle body area | Battery terminals & fuse links |
| Exposed wiring harness connectors | Any exposed sensor connectors |

**No pressure washer in the engine bay — hose only, low pressure, back to front.**

### Engine Bay Steps
1. Confirm fully cool
2. Cover sensitive components (bag + tape)
3. Pre-rinse with low-pressure hose (back → front)
4. Apply AllClean (5:1–10:1) or MultiX (10:1) — 3:1 for heavy grease
5. Agitate with detail brushes
6. Rinse (back → front, low pressure)
7. Remove covers → blow dry with leaf blower → air 10–15 min
8. Dress plastics, hoses & rubber with trim dressing (RTU, apply thin)

## 🛠️ Tool Quick Reference

### Wheel Tools
- **EZ Detail Big Brush** — face, spokes, outer barrel
- **EZ Detail Little Brush** — tight spoke gaps, lug nut pockets
- **Autofiber Barrel Blade** — inner barrel past the lip
- **Detail brushes** — lug nuts, caliper fingers, crevices

### Tire Tools
- **Stiff tire scrub brush** — ReTyre agitation (pre-wash only)
- **Foam applicator pads (Darkside — dedicated)** — never used for anything else

### Glass Tools
- **Exterior MF glass towels** — exterior glass only, never mixed with interior
- **Interior MF glass towels** — interior glass only
- **Invisible Glass Reach & Clean Tool (95160)** — interior windshield & rear screen
  - Need 2–4 heads minimum: 1 damp clean pass, 1 dry buff pass per pane

### Sprayers
- **IK E Foam 12 Pro** (6L usable) — Reset pre-foam, BH Touchless, HydrO2 Foam
- **IK E Foam 2** (1.25L usable) — MWC on wheels, engine bay APC

## 📁 Editing This Plan

All data lives in **`wash-plan.json`** — edit that file to update products, steps, dilutions, or tools.  
The viewer (`index.html`) reads the JSON and renders automatically — no HTML editing needed.

**To add a chemical:** Add an entry to the `chemicals` array in `wash-plan.json`.  
**To add a step:** Add an entry to `washSteps.normal` or `washSteps.maintOnly`.  
**To update a dilution:** Find the chemical by name and edit its `dilutions` array.  
**To add a tool:** Add an entry to the `tools` array.

Publish to GitHub Pages (repo Settings → Pages → main branch / root) to get a live URL for phone access.
